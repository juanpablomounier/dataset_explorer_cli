## Welcome to DATASET_EXPLORER_CLI

This application loads a CSV dataset and generates insights and visualization.



## Structure

DATASET_VISUAL_EXPLORER/
│
├── datasets/
│   └── CSV files
├── outputs/
│   └── Figure files
│
├── main.py
│   └── Orchestrates the application workflow
│
├── loader.py
│   └── Loads and validates CSV datasets
│
├── explorer.py
│   └── Contains the DataExplorer class
│
└── visualizer.py
    └── Displays summarized results and insights

## Installation

pip install -r requirements.txt



## Usage

python main.py
